Manufacturing Production & Rejection Analysis Dashboard
Project Overview

This project analyzes manufacturing production and rejection data to identify quality issues and improve operational efficiency.

Business Problem

The manufacturing process was experiencing high rejection quantities across employees, machines, and departments. The objective was to identify key factors contributing to wastage and provide recommendations for improvement.

Key Insights
Machine C007 generated high rejection volumes.

Certain employees consistently recorded higher rejection quantities.

Recommendations

Investigate high-rejection machines and employees.
